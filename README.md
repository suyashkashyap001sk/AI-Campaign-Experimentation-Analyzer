# AI-Assisted Campaign Experimentation Analyzer

## Overview

The AI-Assisted Campaign Experimentation Analyzer is a Python-based analytics tool designed to automate marketing A/B test analysis.

It evaluates campaign creative performance using statistical significance testing, identifies winning variants, analyzes segment-level performance, and generates automated experiment insights.

This project demonstrates how marketing experimentation workflows can be automated using Python and statistical methods.

---

## Business Problem

Marketing teams frequently run A/B tests to evaluate campaign creatives, messaging, and targeting strategies.

However, analyzing experiment results manually using spreadsheets can be time-consuming and prone to errors. Analysts must calculate funnel metrics, compare performance between test and control groups, and determine statistical significance before making decisions.

This project automates the campaign experimentation workflow by providing a Python-based analytics tool that:

- Calculates campaign funnel metrics  
- Performs statistical significance testing  
- Identifies winning creatives  
- Detects high-performing customer segments  
- Generates automated experiment insights and recommendations  

---

## Key Features

- Automated campaign funnel metric calculation  
- A/B test evaluation using statistical significance testing (Z-Test)  
- Creative-level performance comparison  
- Segment-level performance insights  
- Automated experiment readout and recommendations  
- Interactive web application built with Streamlit  

---

## Experiment Analysis Workflow

The tool follows the typical marketing experimentation workflow:

1. Load campaign performance dataset  
2. Calculate marketing funnel metrics  
3. Aggregate performance by creative and segment  
4. Perform statistical significance testing using Two-Proportion Z-Test  
5. Detect winning creatives  
6. Identify high-performing segments  
7. Generate automated experiment insights  

---

## Campaign Funnel Metrics

The system automatically calculates key marketing funnel metrics:

| Metric | Formula |
|------|------|
| Open Rate | Open / Send |
| Click Rate | Click / Send |
| Response Rate | Applications / Send |
| App Rate | Applications / Click |
| Offer Rate | Offers / Applications |
| Take Rate | Listings / Offers |
| LPM | Listing Amount / Send |
| ALS | Listing Amount / Listings |
| Unsub Rate | Unsubscribes / Send |

---

## Statistical Testing

The experiment analyzer evaluates creative performance using:

Two-Proportion Z-Test

Confidence Level: **90%**

This determines whether performance differences between **Test and Control creatives** are statistically significant.

## Web Application

The project includes an interactive Streamlit application that allows users to:

- Upload campaign datasets  
- Automatically analyze A/B tests  
- Identify winning creatives  
- Detect top performing segments  
- Generate experiment insights  

Run the web app locally:

```
streamlit run app.py
```

The app will open in your browser:

```
http://localhost:8501
```
