# AI Campaign Experimentation Analyzer

This project automates marketing campaign A/B test analysis.

Features:
- Campaign funnel metrics calculation
- Statistical significance testing (90% confidence)
- Creative winner detection
- Segment performance analysis
- AI-style insights generation
- Interactive campaign Q&A assistant

Technologies:
- Python
- Pandas
- Statistical testing (Z-test)
- Jupyter Notebook

Use Case:
Marketing analysts can upload campaign datasets and automatically detect winning creatives and optimization opportunities.

Example Questions the AI Assistant Can Answer:
- Which creative won the test?
- Which segment performs best?
- Is the experiment statistically significant?
- What campaign recommendations should we follow?
